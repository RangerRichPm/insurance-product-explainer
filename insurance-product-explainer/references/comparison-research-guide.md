# 竞品对比调研指南
# 适用于健康险 / 医疗险 / 重疾险

---

## 一、调研目标

从公开渠道找到 3-5 款与待分析产品同类型的竞品，
提炼各自的核心优劣势，形成可客户展示的对比分析。

---

## 二、搜索策略

### 第一步：确定产品赛道

先判断待分析产品属于哪个细分赛道：

| 产品类型 | 细分赛道关键词 |
|---------|--------------|
| 百万医疗险 | 百万医疗险、百万医、医疗险评测 |
| 中端医疗险 | 中端医疗险、中端医疗、特需医疗险 |
| 高端医疗险 | 高端医疗险、BUPA、MSH、安盛 |
| 消费型重疾险 | 消费型重疾险、定期重疾、重疾险评测 |
| 储蓄型重疾险 | 储蓄型重疾险、终身重疾、返还型重疾 |
| 少儿重疾险 | 少儿重疾险、儿童重疾险 |

### 第二步：搜索竞品信息

按优先级使用以下搜索方式：

**优先级1：微信搜一搜（最关键）**
- 搜索词示例：
  - `「产品名」 对比`
  - `「产品名」 评测`
  - `「赛道关键词」 2024 推荐`
  - `「赛道关键词」 哪个好`
- 重点看：保险公众号文章（蜗牛保险、深蓝保、奶爸保、慧择等）
- 这些文章通常有详细的产品对比表格

**优先级2：网页搜索**
- 搜索词示例：
  - `「产品名」 insurance review`
  - `「产品名」 测评 知乎`
  - `「赛道关键词」 性价比 排名`
- 重点看：知乎回答、保险测评网站、代理人个人博客

**优先级3：官方渠道**
- 保险公司官网产品页面
- 保险行业协会产品备案信息
- 各平台在售产品列表（支付宝保险、微信微保、水滴保等）

### 第三步：锁定 3-5 款竞品

从搜索结果中，选出与待分析产品：
- **同赛道**（都是百万医疗 / 都是消费型重疾）
- **同期在售**（不要拿已经停售的产品来比）
- **受众重叠**（目标客户群体相似）

记录竞品名称 + 保险公司，格式：
```
竞品1：XX保险 - 保险公司A
竞品2：YY保 - 保险公司B
...
```

## 三、对比维度

根据产品类型，选择对应的对比维度：

### 医疗险对比维度

| 维度 | 说明 | 客户关心度 |
|-----|------|----------|
| 保证续保年限 | 6年/20年/无保证续保 | ⭐⭐⭐⭐⭐ |
| 免赔额 | 1万/5千/0免赔 | ⭐⭐⭐⭐⭐ |
| 报销范围 | 医保内/医保内外/含靶向药 | ⭐⭐⭐⭐ |
| 保费 | 同条件每年多少钱 | ⭐⭐⭐⭐⭐ |
| 质子重离子 | 有/无、报销比例 | ⭐⭐⭐ |
| 院外靶向药 | 有/无、药品清单 | ⭐⭐⭐⭐ |
| 垫付/直付 | 有/无 | ⭐⭐⭐ |
| 健康告知 | 严/松 | ⭐⭐⭐⭐ |

### 重疾险对比维度

| 维度 | 说明 | 客户关心度 |
|-----|------|----------|
| 保费 | 同条件每年多少钱 | ⭐⭐⭐⭐⭐ |
| 重疾赔付 | 1次/多次、间隔期 | ⭐⭐⭐⭐ |
| 轻症赔付比例 | 20%/30%/45% | ⭐⭐⭐ |
| 60岁前额外赔 | 有/无、比例 | ⭐⭐⭐⭐ |
| 癌症二次赔付 | 有/无、间隔期 | ⭐⭐⭐⭐ |
| 身故责任 | 含/不含/可选 | ⭐⭐⭐ |
| 保费豁免 | 轻症后免交/无 | ⭐⭐⭐ |

## 四、信息来源可信度评估

| 来源 | 可信度 | 备注 |
|-----|-------|------|
| 保险公司官网 | ⭐⭐⭐⭐⭐ | 最权威，但可能只说好的 |
| 保险行业协会备案 | ⭐⭐⭐⭐⭐ | 条款原文，最准确 |
| 专业保险测评号 | ⭐⭐⭐⭐ | 相对客观，但可能有倾向性 |
| 代理人个人文章 | ⭐⭐⭐ | 可能有 bias，需交叉验证 |
| 知乎高赞回答 | ⭐⭐⭐ | 看回答者的专业背景 |
| 产品宣传页 | ⭐⭐ | 只说亮点，需核实 |

**原则：关键数据（保费、保额、免赔额）必须有两个以上来源交叉验证。**

---

## 五、输出竞品对比时的注意事项

1. **不做绝对定性**：不说「A产品比B产品好」，而是「A在X方面更有优势，B在Y方面更有优势」
2. **说明适用场景**：「如果你看重保证续保，选A；如果你看重0免赔，选B」
3. **标注信息来源和时间**：「数据来源：各产品官网，截至2024年X月」
4. **坦诚说明局限性**：「以下对比基于公开信息整理，具体以保险公司最新条款为准」
