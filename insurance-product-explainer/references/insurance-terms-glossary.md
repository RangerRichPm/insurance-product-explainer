# 保险术语白话翻译表
# 适用于健康险 / 医疗险 / 重疾险

---

## 一、保额与保障类

| 专业术语 | 白话解释 |
|---------|---------|
| 保额 | 出事最多能赔你多少钱 |
| 保障期限 | 这份保险保你多长时间 |
| 等待期（观察期） | 买完保险后，这段时间出事不赔（防止带病投保） |
| 犹豫期 | 买完保险后某段时间内可以无理由退保，全额退保费 |
| 保障责任 | 这份保险到底保什么 |
| 主险 / 附加险 | 主险是主干，附加险是额外加的保障（像汉堡加配料） |
| 保险期间 | 合同有效期，过了这个期限保障就结束了 |

---

## 二、保费与缴费类

| 专业术语 | 白话解释 |
|---------|---------|
| 保费 | 你每年/每月要交多少钱 |
| 缴费期限 | 你要连续交多少年保费 |
| 趸交 | 一次性把全部保费交完 |
| 期交 | 分多年缴费（比如交20年） |
| 保费豁免 | 发生某种情况后，后面的保费不用交了，但保障继续有效 |
| 现金价值 | 如果现在退保，保险公司能退给你多少钱 |

---

## 三、理赔与报销类

| 专业术语 | 白话解释 |
|---------|---------|
| 理赔 | 出事之后找保险公司赔钱 |
| 免赔额（起付线） | 花销低于这个数，保险公司不赔；超过的部分才开始赔 |
| 赔付比例（报销比例） | 保险公司帮你承担多少百分比的费用 |
| 报销范围 | 哪些费用可以找保险公司报 |
| 合理且必要 | 保险公司认可的、治疗确实需要的医疗费用 |
| 既往症 | 买保险之前就已经得的病（通常不赔） |
| 先行垫付 | 住院时保险公司先帮你出钱，不用自己掏 |

---

## 四、医疗险特有术语

| 专业术语 | 白话解释 |
|---------|---------|
| 住院医疗 | 住院期间产生的医疗费用 |
| 特殊门诊 | 不用住院、但在门诊做的特殊治疗（如化疗、透析） |
| 门诊手术 | 在门诊就能完成的小手术 |
| 住院前后门急诊 | 住院前一段时间和出院后一段时间的门急诊费用 |
| 直付 / 垫付 | 看病时保险公司直接跟医院结账，不用自己先掏钱 |
| 质子重离子 | 一种治疗癌症的高端技术，一般医疗险不包，好的才包 |
| 院外靶向药 | 医院开不出来的抗癌药，去外面药店买（好产品会报销这个） |
| 保证续保 | 续保时保险公司不能因为你的健康状况变化或理赔过而拒保或涨价 |
| 续保年龄上限 | 这份保险最多能续保到多少岁 |

---

## 五、重疾险特有术语

| 专业术语 | 白话解释 |
|---------|---------|
| 重大疾病 | 合同约定的严重疾病（如癌症、心梗、脑中风等） |
| 轻症 / 中症 | 比重疾轻一些的病，但也不轻（好的产品轻症也能赔） |
| 重疾多次赔付 | 得了一次重疾赔完，以后还得别的重疾还能再赔 |
| 癌症二次赔付 | 癌症治好后又复发或转移，还能再赔一次 |
| 心脑血管二次赔付 | 心梗/脑中风治好后又犯，还能再赔一次 |
| 疾病终末期 | 医生判定活不太久了，保险公司提前赔 |
| 身故责任 | 没得重疾就死了，赔不赔、怎么赔 |
| 投保人豁免 | 给你买保险的那个人出事了，你以后的保费不用交了 |

---

## 六、除外与限制类

| 专业术语 | 白话解释 |
|---------|---------|
| 免责条款（除外责任） | 这些情况保险公司明确不赔 |
| 健康告知 | 买保险前要如实回答的健康问题（答错了可能影响理赔） |
| 职业限制 | 有些工作危险，保险公司不保或加钱才保 |
| 医院限制 | 必须在哪些医院看病才能报销（一般是二级以上公立医院） |
| 医保范围内 / 外 | 医保能报的 vs 医保不能报的（好的医疗险两者都报） |

---

## 七、常用对比指标（客户最关心的）

| 指标 | 客户关心的点 |
|-----|-------------|
| 保额够不够 | 生病了赔的钱够不够用 |
| 保费贵不贵 | 每年交的钱是不是划算 |
| 免赔额高不高 | 自己要先掏多少钱才能开始报销 |
| 什么病都能保吗 | 有没有什么病不保 |
| 理赔方不方便 | 出事之后赔钱麻不麻烦 |
| 能保多久 | 是保一年还是要每年续保 |
| 健康要求严不严 | 身体有小毛病还能买吗 |
