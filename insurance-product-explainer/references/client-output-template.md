# 客户输出模板
# 适用于健康险 / 医疗险 / 重疾险

---

## 模板使用说明

本文件包含两个版本的模板：
1. **Markdown 底稿版**：结构完整，用于内部整理和存档
2. **微信精简版**：段落短、emoji 适度，适合直接发给客户

根据客户需求，选择对应版本输出，或先出底稿版再转微信版。

---

## 三、输出检查清单

输出前，逐条确认：

- [ ] 所有专业术语都已翻译成白话
- [ ] 亮点不超过5个（多了客户记不住）
- [ ] 竞品对比数据有来源（标注来源）
- [ ] 除外责任已坦诚说明（不隐瞒）
- [ ] 保费数字已核对（至少2个来源）
- [ ] 微信版段落短（每段不超过3行）
- [ ] 微信版适当使用 emoji（不过度）
- [ ] 最后有免责声明（以条款为准）
