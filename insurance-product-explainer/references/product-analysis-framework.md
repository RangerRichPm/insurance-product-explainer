# 保险产品分析框架
# 适用于健康险 / 医疗险 / 重疾险

---

## 使用说明

收到保险产品文档后，按以下框架逐项提取信息。
提取完后，用「白话翻译表」将专业术语转成客户能懂的语言。

---

## 一、医疗险（百万医疗 / 中端医疗 / 高端医疗）分析维度

### 基础信息
- 产品名称、保险公司、产品类型
- 投保年龄范围、职业限制
- 保障期限（1年 / 保证续保X年）
- 等待期（一般30天）

### 保障责任（核心）
- 一般医疗保险金保额（住院 + 特殊门诊 + 门诊手术 + 住院前后门急诊）
- 免赔额（常见1万，有的0免赔）
- 报销比例（经医保 vs 未经医保，100% vs 60%）
- 报销范围（医保内 only / 医保内+外 / 含院外靶向药）
- 质子重离子是否覆盖、报销比例
- 特殊门诊具体包含哪些（化疗、透析、抗排异等）
- 住院前后门急诊覆盖天数（前7后7 / 前30后30）

### 加分项（是否为亮点）
- 保证续保年限（6年 / 20年 / 终身）
- 续保年龄上限（80岁 / 100岁）
- 是否含院外靶向药（及药品清单数量）
- 是否有先行垫付 / 直付功能
- 是否含门诊（一般百万医疗不含，含的是中端/高端医疗）
- 重疾0免赔（确诊重疾后免赔额降为0）
- 家庭投保优惠（多人投保打折）

### 保费（核心对比项）
- 有社保版本 vs 无社保版本的保费
- 0岁 / 30岁 / 50岁 保费举例（以表格呈现）

### 除外责任（需告知客户）
- 既往症不赔
- 整形、生育、牙科一般不含
- 酒驾、犯罪等免责条款

### 健康告知严苛度
- 是否需要体检
- 常见异常（甲状腺结节、乳腺结节、高血压）是否能投

## 二、重疾险分析维度

### 基础信息
- 产品名称、保险公司、产品类型（消费型 / 储蓄型 / 返还型）
- 投保年龄、保障期限（定期的如70岁 / 终身）
- 等待期（一般90天或180天）

### 疾病保障（核心）
- 重疾：保多少种、怎么赔（赔1次 / 多次赔）
- 轻症：保多少种、赔付比例（20% / 30% / 45%）
- 中症：保多少种、赔付比例（50% / 60%）
- 是否含「重疾额外赔付」（如60岁前确诊额外赔50%）
- 癌症二次赔付：间隔期多长（3年 / 5年）、赔付比例
- 心脑血管二次赔付：同上

### 身故责任
- 含身故（赔保额）/ 不含身故（赔现金价值）/ 可选

### 保费（核心对比项）
- 50万保额、30年缴费、保终身 的保费（分男女）
- 若含60岁前额外赔，展示实际杠杆

### 加分项（是否为亮点）
- 轻症/中症赔付后重疾保额是否增加
- 是否含保费豁免（轻症/中症/重疾后免交后续保费）
- 绿通服务（挂号、住院安排等）
- 等待期内出险的处理方式（友好 vs 不友好）

### 除外责任
- 遗传性疾病、艾滋病、自杀等通常不赔
- 重点提示客户

## 三、信息提取 Checklist

收到产品文档后，逐条确认是否已提取：

**医疗险**
- [ ] 产品名称 + 保险公司
- [ ] 投保年龄范围
- [ ] 保证续保年限
- [ ] 一般医疗保额
- [ ] 免赔额
- [ ] 报销范围（医保内/外/靶向药）
- [ ] 质子重离子
- [ ] 保费（至少3个年龄段）
- [ ] 除外责任要点
- [ ] 健康告知严苛度

**重疾险**
- [ ] 产品名称 + 保险公司
- [ ] 重疾种类 + 赔付次数
- [ ] 轻症/中症种类 + 赔付比例
- [ ] 癌症二次赔付（有/无 + 间隔期）
- [ ] 60岁前额外赔付（有/无 + 比例）
- [ ] 身故责任
- [ ] 保费（男女各一例）
- [ ] 除外责任要点

## 四、亮点提炼原则

提炼亮点时，遵循以下原则：

1. **用客户视角说话**：不要说「含质子重离子责任」，要说「癌症高端治疗也能报销」
2. **数字说话**：「30岁男性，一年仅需XXX元」比「保费低廉」更有说服力
3. **对比才有意义**：「保证续保20年」要解释为「这20年里，无论你得没得病、理赔过没，都能续保——很多产品做不到」
4. **只说最重要的3-5个亮点**：不要堆砌，客户记不住
5. **坦诚说明限制**：不隐瞒除外责任，反而建立信任
